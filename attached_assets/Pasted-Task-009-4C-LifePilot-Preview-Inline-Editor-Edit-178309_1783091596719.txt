Task-009.4C：完成「我的分類」管理功能

目前 LifePilot 已完成：

- 我的頁面骨架
- 分類管理入口
- Preview Inline Editor
- EditPage
- 事項類型 Type
- 我的分類 Category
- LocalStorage 資料保存

目前「我的 → 分類管理」仍顯示「即將推出」。

本次任務目標：

完成真正可使用的分類管理功能。

注意：

「事項類型」與「我的分類」是不同概念。

事項類型 Type：
由系統固定，用來決定資料結構與欄位。

我的分類 Category：
由使用者管理，用來整理事項。

本次只做 Category 分類管理。

不要允許使用者新增自訂 Type。

────────────────

一、開啟分類管理頁

點擊：

我的
→ 分類管理

進入獨立的「分類管理」頁面。

頁面顯示：

分類管理
建立與管理你的個人分類

內容分為：

1. 我的分類
2. 預設分類

────────────────

二、系統預設分類

預設提供：

- 工作
- 家庭
- 小孩
- 個人
- 生活
- 重要
- 其他

預設分類可以：

- 使用
- 調整排序
- 隱藏

預設分類暫時不要永久刪除。

如果隱藏：

舊事項仍保留原 category，
只是新增與編輯時不再出現在主要選單。

────────────────

三、自訂分類

使用者可以新增自己的分類。

例如：

- 寵物
- 健身
- 車輛
- 客戶 A
- 旅遊
- 房租

提供按鈕：

＋ 新增分類

點擊後開啟簡單 Modal 或 Drawer。

欄位：

分類名稱
分類顏色
分類圖示

按鈕：

取消
建立分類

────────────────

四、分類資料結構

請建立獨立 Category 資料結構。

至少包含：

id
name
color
icon
enabled
isSystem
sortOrder
createdAt

請依目前 TypeScript 架構定義正確型別。

分類資料必須保存到 LocalStorage。

重新整理頁面後不能消失。

不要與 Reminder Data 混在同一個 Storage Key。

請使用獨立且清楚的 Storage Key。

────────────────

五、分類卡片

分類管理頁每個分類顯示成簡潔卡片或列表項目。

顯示：

圖示
分類名稱
顏色標記

自訂分類提供：

編輯
停用
刪除

系統分類提供：

隱藏
恢復顯示

不要讓系統分類永久刪除。

────────────────

六、編輯分類

點擊自訂分類的「編輯」後，可以修改：

分類名稱
分類顏色
分類圖示

儲存後：

Preview Inline Editor 的「我的分類」選單立即更新。

EditPage 的「我的分類」選單立即更新。

不需要重新整理頁面。

────────────────

七、新增分類後立即可使用

例如使用者正在 Preview Inline Editor：

我的分類
→ 打開選單

選單內容顯示目前 enabled 的分類。

選單底部加入：

＋ 新增分類

點擊後可以直接建立分類。

例如建立：

寵物

完成後：

1. 寵物立即出現在分類選單。
2. 自動選取新建立的「寵物」分類。
3. 不重新呼叫 AI。
4. 不離開目前編輯中的事項。
5. 不遺失目前 Preview 修改內容。

EditPage 的分類選單也要支援同樣流程。

────────────────

八、分類刪除安全機制

如果自訂分類完全沒有被任何 Reminder 使用：

可以刪除。

如果分類已被 Reminder 使用：

不要直接刪除。

顯示提示：

「此分類仍有事項使用中」

提供：

取消
停用分類

本次不要自動批次修改舊 Reminder 的 category。

────────────────

九、分類選單共用

請避免：

Preview Inline Editor 維護一份分類陣列，
EditPage 又維護另一份，
分類管理頁再維護第三份。

請建立共用的 Category Store 或 Category Hook。

三個地方使用同一份資料來源：

1. Category Management Page
2. Preview Inline Editor
3. Reminder EditPage

新增、修改、隱藏分類後，
所有分類選單應立即同步。

────────────────
